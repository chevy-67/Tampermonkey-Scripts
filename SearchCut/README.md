
# SearchCut

This script is used to search for a selected text in a page with custom search query. It reduces the manual work of copy pasting unkown words or terminology to search for its meaning

## Usage
- Select a text that needs to be searched.
- Press the key ```Windows Key```+```x```

## Customization
- You can change the key as your convenient in the script
- Also you can change the query to be searched for the selected text ie,
    * ```What is ```
    * ```Explain```
    * ```Definition for```


## Keys with their key-code

| Property         | Meaning                                                |
| ---------------- | ------------------------------------------------------ |
| `event.ctrlKey`  | CTRL key is pressed                                    |
| `event.shiftKey` | SHIFT key is pressed                                   |
| `event.altKey`   | ALT key is pressed                                     |
| `event.metaKey`  | META key (Windows key / Command key on Mac) is pressed |
